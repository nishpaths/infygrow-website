# InfyGrow Website

A modern, responsive website for InfyGrow - a financial services and insurance company (also my dad's buisness lol). Built with React and Vite, featuring a sleek design and comprehensive service information.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Mobile-First Navigation**: Enhanced hamburger menu with smooth animations for mobile and tablet users
- **Service Pages**: Comprehensive information about various financial and insurance services including:
  - Wealth Management
  - Health Insurance
  - Travel Insurance
  - Life Insurance
  - Retirement Planning
  - Long-Term Care
  - Final Expenses
  - Real Estate
  - Investments
  - College Savings
  - Tax Savings
  - Business Opportunities
  - Visitors Insurance
  - Will & Trust Services
- **Team Page**: Showcase of the InfyGrow team
- **Contact Form**: Easy way for clients to get in touch
- **Why Us Section**: Information about InfyGrow's value proposition

## Key Features

### Responsive Navigation

The navbar includes:
- Sticky navigation bar
- Hamburger menu for mobile/tablet devices
- Smooth slide-in animations
- Backdrop overlay
- Auto-close on navigation
- Services dropdown with mobile-friendly interactions

### Color Scheme

- Primary background: Black (#000000)
- Accent color: Gold (#bab675)
- Text: White with gray hover states

## Development

The project uses:
- **Vite** for fast HMR (Hot Module Replacement) during development
- **ESLint** for code quality and consistency
- **Tailwind CSS** for utility-first styling

## Browser Support

Modern browsers that support ES6+ features:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
